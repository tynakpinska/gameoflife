# gameoflife

Live Demo
https://game-of-life-front.herokuapp.com/

Used technologies:
- HTML,
- CSS, CSS modules,
- JavaScript,
- React,
- Redux,
- NodeJS,
- express,
- postgresql,
- knex.js,
- bcrypt,
- json web token.

Features:
- register,
- log in,
- add, edit, save your challenges for the day,
- toggle challenges,
- see how much time you have left till the end of the day,
- check your streak,
- add, edit, save your goals,
- add, edit, save your profile picture
